# Sistema de Gestión Clínica Hospitalaria

Este proyecto consiste en el desarrollo de un sistema de gestión hospitalaria que permite administrar información clave de pacientes, médicos, consultas y servicios médicos, 
facilitando la organización de procesos dentro de una clínica u hospital.

---

# Objetivo
- Diseñar e implementar una aplicación que permita gestionar de manera estructurada los datos de un entorno hospitalario, aplicando principios de programación orientada a objetos y organización de software.

---

# Funcionalidades principales
- Registro de pacientes
- Gestión de médicos
- Administración de consultas
- Control de servicios hospitalarios
- Relación entre entidades (paciente ↔ médico ↔ consulta)

---

# Tecnologías utilizadas
- Java
- Programación Orientada a Objetos (POO)
